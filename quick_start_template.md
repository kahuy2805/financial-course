## üÜò Troubleshooting

**If the output isn't what you expected:**
1. Review your template - was anything unclear?
2. Provide specific feedback on what to adjust
3. Use the meta-prompt system as reference for quality standards
4. Request specific examples before full creation

**If lessons are too complex/simple:**
- Adjust "Current Knowledge Level" in learner profile
- Specify desired "Lesson Length"
- Request example lesson for review first

**If structure doesn't fit your needs:**
- Propose alternative organization
- Reference successful course structures you like
- Specify exact chapter/lesson breakdown you prefer
