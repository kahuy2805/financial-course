# COURSE CREATION REQUEST
## Stock Trading: Zero to Hero

---

## LEARNER PROFILE
- **Current Knowledge Level**: Complete beginner (zero experience)
- **Age Range**: General audience (adaptable)
- **Primary Goal**: Personal wealth building through smart investing
- **Time Availability**: Flexible/self-paced learning
- **Trading Capital**: $10,000 - $25,000 starting capital

## FOCUS AREAS
- **Market Type**: US Stocks primarily (can expand to other markets later)
- **Trading Style**: Mixed approach (day trading, swing trading, long-term investing)
- **Analysis Preference**: Both technical and fundamental analysis
- **Risk Tolerance**: Conservative to moderate with emphasis on capital preservation

## COURSE SCOPE
- **Depth Level**: Comprehensive - building solid foundation to advanced mastery
- **Estimated Duration**: Flexible/self-paced (learners progress at their own speed)
- **Platform Focus**: Platform-agnostic (applicable to any brokerage)
- **Include**:
  - [x] Technical analysis charts and patterns
  - [x] Fundamental analysis and financial statements
  - [x] Options trading (advanced level)
  - [x] Risk management and position sizing
  - [x] Trading psychology
  - [x] Portfolio management
  - [x] Tax considerations
  - [x] Broker selection
  - [x] Paper trading practice
  - [x] Real money management strategies

## CUSTOMIZATION
- **Teaching Style**: Balanced - mix of clear explanations, real-world examples, visual descriptions, and practical applications
- **Assessment Type**: Mixed - quizzes for knowledge check, case studies for application, and practical exercises
- **Mentorship Format**: 1-on-1 guidance framework with reflection prompts, Q&A support, and personalized learning pathways
- **Special Requirements**:
  - Easy to read and understand for complete beginners
  - Build progressively from zero to hero
  - Include both theory and real-world practical application

## OUTPUT PREFERENCES
- **File Organization**: By level (Foundation → Intermediate → Advanced)
- **Lesson Length**: Medium 15-20 minutes reading time
- **Practice Frequency**: After each chapter with cumulative practice exercises
- **Include Answer Keys**: Yes, with detailed explanations and reasoning

## CONSTRAINTS & BOUNDARIES
- **Topics to Avoid**:
  - Penny stocks and micro-cap speculation
  - Excessive leverage and high-risk strategies
  - Get-rich-quick schemes or unrealistic promises

- **Topics to Emphasize**:
  - **Risk management** - Capital preservation as priority #1
  - **Trading psychology** - Emotional discipline and behavioral finance throughout
  - **Tax considerations** - Understanding tax implications of trading
  - **Practical application** - Real-world examples and actionable strategies

- **Regulatory Context**: US markets and regulations (SEC, FINRA, IRS guidelines)
- **Budget Assumptions**: Use $10K-$25K in examples for position sizing and portfolio allocation

## ADDITIONAL NOTES
- **Structure**: Clear separation between Foundation and Advanced levels
  - Foundation: Core concepts, basics, getting started
  - Intermediate: Building skills, analysis techniques
  - Advanced: Professional strategies, portfolio management

- **Mentorship**: Include 1-on-1 mentorship framework with:
  - Self-assessment tools
  - Personalized learning paths based on goals/risk tolerance
  - Coaching conversation guides
  - Progress tracking mechanisms

- **Practical Focus**: Balance theory with real-world application
  - Include recent market examples and case studies
  - Show both winning and losing trades (learning from mistakes)
  - Address common beginner mistakes in each section

- **Progressive Learning**: Build from zero to hero
  - Complete beginners should be able to start with no prior knowledge
  - Advanced sections should prepare learners for professional-level trading
  - Clear progression path showing skill development

- **File Format**:
  - All lessons in separate markdown files
  - Master index linking to all chapters and lessons
  - Clear navigation between lessons
  - Professional book-like structure

---

**Ethical Foundation**:
- Include mandatory risk disclaimers
- Set realistic expectations (most traders lose money initially)
- Emphasize continuous learning and patience
- Encourage consulting licensed professionals for personalized advice
