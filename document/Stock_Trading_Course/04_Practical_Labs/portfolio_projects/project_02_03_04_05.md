## Project 05: Trading & Active Portfolio

### Profile
- **Goal**: Active returns through trading
- **Time Horizon**: Ongoing (trade-by-trade basis)
- **Risk Tolerance**: Very high (40%+ swings OK)
- **Target Return**: 15-25% annually (with skill)
- **Volatility**: 30%+ (highly variable)
- **Time Commitment**: 10+ hours/week

### Allocation Framework
```
Core Holdings (50%):    Index funds + dividend stocks (no trading)
Tactical Trades (30%):  Swing trades, sector rotations
Options/Derivatives (10%): Covered calls, spreads, directional bets
Cash (10%):            Dry powder for opportunities
```

### Core Holdings (Non-Traded)
```
TICKER  | TYPE              | ALLOCATION | PURPOSE
--------|-------------------|------------|------------------
SPY     | US Market         | 30%        | Core, don't trade
SCHD    | Quality Divs      | 15%        | Income, stability
VXUS    | International     | 5%         | Diversification
VMFXX   | Cash              | 10%        | Opportunities
```

### Tactical Allocations (Actively Traded)
```
Position Type | Allocation | Example
─────────────|────────────|──────────────────────
Swing Trades | 15%        | MSFT breakout, 3-4 week hold
Sector Trades| 10%        | Energy rally, 2-3 month hold
Individual   | 5%         | Deep value, 6-month hold
```

### Options Strategies (If Trading Options)
```
Strategy          | Allocation | Max Risk | Use Case
─────────────────|────────────|----------|──────────────
Covered Calls    | 5%         | Limited  | Generate income
Cash-Secured Puts| 3%         | Defined  | Enter positions
Bull Call Spreads| 2%         | Defined  | Directional bets
Iron Condors     | 0-2%       | Defined  | Income generation
```

### Performance Expectations
```
With skill (top 10% of traders):
- Return: 18-25% annualized
- Volatility: 30%+
- Worst year: -20% to -40%
- Best year: +50% to +100%

Without skill (average trader):
- Return: 4-6% (underperform market)
- Volatility: 30%+ (much higher risk)
- Worst year: -40% to -60%
- Best year: +15% to +25%
```

### Rules for This Portfolio
```
1. Core 50% never touched (buy and hold)
2. Never trade core holdings
3. Tactical 30%: Max 5% per position
4. Options 10%: Defined risk only
5. Cash 10%: Minimum, never go to zero
6. Stop losses: Mandatory on all trades
7. Rebalancing: Quarterly (rebalance core if needed)
8. Emotions: Leave if angry, afraid, greedy
```

### Monitoring Requirements
```
Daily: Check open positions, news
Weekly: Evaluate tactical positions, make new trades
Monthly: Tally profits/losses, tax planning
Quarterly: Rebalance core, review strategy
Annually: Performance audit, rule refinement
```

### Is This For You?
✓ Experience: 3+ years investing minimum
✓ Time: 10+ hours/week available
✓ Temperament: Can handle losses calmly
✓ Edge: Have an actual trading system
✓ Capital: Can afford to lose some
✓ Knowledge: Understand technical analysis, options
✓ Discipline: Can follow rules always

### Getting Started
```
Month 1: Set up core portfolio (50% allocation)
Month 2: Paper trade the tactical portion
Month 3: Small real trades ($500-1,000 positions)
Month 4-6: Refine strategy, track results
Month 6+: Scale up if winning, or retreat if losing
```

### Important Warnings
```
⚠ Trading is hard: 95% of traders underperform markets
⚠ Time consuming: Need 10+ hours/week minimum
⚠ Psychological: Requires emotional discipline
⚠ Costs: Commissions and taxes eat gains
⚠ Taxes: Short-term capital gains taxed at ordinary income rate
⚠ Drawdowns: Can lose 30-40% in bad periods
```

## Quick Decision Tree

**Pick Conservative If:**
- Age: 55+
- Risk tolerance: Low
- Time horizon: 10+ years
- Income needs: Yes
- Risk appetite: Can't handle 20% declines

**Pick Balanced If:**
- Age: 40-55
- Risk tolerance: Medium
- Time horizon: 15+ years
- Income needs: Not yet
- Risk appetite: Can handle 15% declines

**Pick Aggressive If:**
- Age: 20-40
- Risk tolerance: High
- Time horizon: 20+ years
- Income needs: No
- Risk appetite: Can handle 40% declines

**Pick Income If:**
- Status: Retired or semi-retired
- Time horizon: 15+ years
- Income needs: Yes, currently
- Risk tolerance: Medium
- Asset level: $250,000+

**Pick Trading If:**
- Experience: 3+ years investing
- Time available: 10+ hours/week
- Temperament: Calm under pressure
- Have an edge: Trading system tested
- Capital: Can afford some losses
